# Images Folder

Положите сюда фотографии тюльпанов.

## Рекомендуемый формат изображений:

- **Формат**: JPG или WEBP
- **Размер**: минимум 800x800 px
- **Соотношение сторон**: 1:1 (квадрат)
- **Размер файла**: до 500 KB

## Название файлов:

Для каждого сорта создайте файл с именем `tulip-[ID].jpg`, где ID - номер сорта.

Примеры:
- `tulip-1.jpg` - Красный барон
- `tulip-2.jpg` - Апельсин
- `tulip-3.jpg` - Белый рай
- и т.д.

Полный список сортов смотрите в файле `script.js` в массиве `products`.
