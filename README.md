# 🌷 Лендинг для продажи тюльпанов

Одностраничный сайт (landing page) для продажи тюльпанов оптом в Челябинске.

## 📁 Структура проекта

```
tulpan/
├── index.html          # Главная HTML-страница
├── styles.css          # Стили (весенняя цветовая гамма)
├── script.js           # JavaScript функционал (фильтры, поиск)
├── images/             # Папка для фотографий тюльпанов
└── README.md           # Этот файл
```

## 🎨 Особенности дизайна

- **Весенняя цветовая гамма**: розовый, персиковый, желтый, зеленый
- **Адаптивный дизайн**: работает на всех устройствах
- **Mobile-first подход**: оптимизировано для мобильных
- **Современные технологии**: HTML5, CSS3, Vanilla JavaScript

## 🌷 Каталог сортов

Каталог содержит более 50 сортов тюльпанов, разбитых на категории:

- **Простые** - классические формы
- **Махровые** - пышные многослойные бутоны
- **Лилиецветные** - с заостренными лепестками
- **Бахромчатые** - с бахромой по краям лепестков
- **Попугайные** - с волнистыми курчавыми лепестками

## 📸 Добавление фотографий

Чтобы добавить фотографии тюльпанов:

1. Поместите файлы в папку `images/`
2. Назовите файлы в соответствии с ID сортов:
   - `tulip-1.jpg` для "Красный барон"
   - `tulip-2.jpg` для "Апельсин"
   - и так далее...

3. Отредактируйте `script.js`, заменив emoji-иконки на реальные изображения:

```javascript
// Было:
<div class="product-card__image">
    <span class="product-card__emoji">${colorEmoji}</span>
</div>

// Станет:
<div class="product-card__image">
    <img src="images/tulip-${product.id}.jpg" alt="${product.name}">
</div>
```

## ✏️ Редактирование информации

### Контактные данные

Отредактируйте в `index.html`:

```html
<!-- Телефон в шапке -->
<a href="tel:+79000000000" class="header__phone">
    <span>+7 (900) 000-00-00</span>
</a>

<!-- Телефон в контактах -->
<a href="tel:+79000000000" class="contact-card__link">+7 (900) 000-00-00</a>

<!-- Адрес -->
<p>Челябинск, [уточнить адрес]</p>
```

### Сорта и цены

Отредактируйте массив `products` в `script.js`:

```javascript
{
    id: 1,
    name: 'Название сорта',
    category: 'simple', // simple, double, lily, fringed, parrot
    color: 'red',       // red, pink, white, yellow, orange, purple, green
    price: 70,          // цена в рублях
    description: 'Описание сорта',
    features: ['Особенность 1', 'Особенность 2'],
    popular: true       // показать звездочку "Популярный"
}
```

## 🚀 Развертывание

### GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта
3. Включите GitHub Pages в Settings
4. Сайт будет доступен по адресу: `https://username.github.io/tulpan/`

### Netlify / Vercel

Просто перетащите папку проекта в Netlify Drop или Vercel.
